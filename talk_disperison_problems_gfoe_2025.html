Intro

Residual diagnostics in GLMM

  • DHARMa approach

Goals

Provide tools for ecologists to better detect and solve the many dispersion problems that may arrise in GLMMs:

  • Heteroscedasticity

  • Zero-inflation

  • “Real” overdispersion

Examples will be. using DHARMa, glmmTMB, and ggeffects packages.

Problem 1: Heteroscedasticity

# wrong model
glmmTMB(observedResponse ~ Environment1 +
          (1|group), 
          family = "poisson", data = heteroData)
heteroRes <- simulateResiduals(heteroModel)
plot(heteroRes)

Many think this is not a problem for GLMMs, but I’m showing here that YES, this is possible in GLMS, for example the poisson model, and that it can create overdispersion - but one that may be solved by modeling the dispersion term, not only changing the distribution to the negative binomial, for example.

Problem 1: Heteroscedasticity

plotResiduals(heteroRes, 
    form=heteroData$Environment1, 
    absoluteDeviation = T)
testDispersion(heteroRes)

    DHARMa nonparametric dispersion test via sd of residuals fitted vs.
    simulated

data:  simulationOutput
dispersion = 1.8953, p-value < 2.2e-16
alternative hypothesis: two.sided
testZeroInflation(heteroRes)

    DHARMa zero-inflation test via comparison to expected zeros with
    simulation under H0 = fitted model

data:  simulationOutput
ratioObsSim = 1.0962, p-value = 0.248
alternative hypothesis: two.sided

Solving heteroscedasticity
